# Brian Strength (Option A: Phone-first PWA)

This project deploys to Vercel and installs on your phone like an app.
Data is stored on-device (localStorage).

## Local run
```bash
npm install
npm run dev
```

## Deploy (Vercel)
1) Push this folder to a GitHub repo
2) Vercel → New Project → import repo → Deploy

## Install on phone
- iPhone Safari: Share → Add to Home Screen
- Android Chrome: Install app / Add to Home screen

## If icons don't update
1) Remove the home screen icon
2) Hard refresh the site in Safari/Chrome
3) Re-add to home screen
